# Helm Visual Identity Guide

## Logo Usage

Primary logo:
Anchor icon on blue gradient.

Clear space:
Minimum equal to anchor ring height.

Never:
- Stretch
- Add heavy drop shadow
- Add glow
- Place on clashing gradients

---

## Background Strategy

Marketing:
Deep navy gradient with subtle radial glow.

App:
Light mode by default.
Dark mode as command deck aesthetic.

---

## Imagery Style

Use:
- Clean macOS screenshots
- Subtle depth
- Blue-focused palette

Avoid:
- Stock photos
- Abstract startup blobs
- Overly colorful gradients

---

## Pro Visual Distinction

Helm Pro elements:
- Gold accent
- Slight elevation
- Subtle premium highlight

Never:
- Change entire app color scheme
- Over-saturate UI

Pro should feel premium, not flashy.

---

## Design Inspiration (Reference Only)

Avast:
- Strong central message
- High contrast CTA

Docker Desktop:
- Clean side navigation
- Focused workspace
- Clear hierarchy

Helm should borrow:
- Clarity
- Contrast
- Structured navigation

Helm should avoid:
- Purple neon
- Overly gamified visuals
